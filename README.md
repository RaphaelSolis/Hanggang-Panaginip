# Hanggang-Panaginip
Dreams, Regrets, and Reality — Turtle Graphics Story Project

A story-driven Python Turtle animation project created for ITE102.
This program combines graphics, dialogue, music, and scene transitions to tell an emotional narrative about love, loss, regret, and acceptance.

Features:

• Multiple animated scenes using Python Turtle
• Custom character/environment drawings
• Dialogue typing effects
• Audio integration using MPV
• Emotional story progression with cinematic transitions
• Original narrative and visual design


Requirements

Before running the project, install the following:

Python 3
Turtle module
Time module
OS module
MPV Media Player (required for audio playback)


How to Run

1. Install Python and MPV.

2. Place the .py file and audio files in the same folder.

3. Run the Python file.

python filename.py
